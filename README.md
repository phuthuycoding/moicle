# Claude Agents Kit

Reusable agents and commands for Claude Code to bootstrap and develop projects faster.

## Install

```bash
git clone https://github.com/yourname/claude-agents-kit.git ~/.claude-kit
~/.claude-kit/install.sh
```

## Structure

```
agents/
├── developers/     # Stack-specific dev agents
└── utilities/      # General utility agents
commands/           # Global commands
templates/          # Project templates
```

## Usage

```
/bootstrap          # Create new project
@code-reviewer      # Review code
@test-writer        # Write tests
```
